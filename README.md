# Medical-Prescription-Safety
An AI-Based System the helps the Doctors and Hospitals detect Prescription related risks before medicine are prescribe to the patient. This System analyzes the drug-drug interactions, overdose risks, age-based warnings, and allergy conflicts. It provides a clear risk score and safety alerts to reduce medical errors and improve patients safety.
# AI-Powered Prescription Safety System

A medical software system that analyzes prescriptions and detects drug interaction risks using AI.

## Tech Stack
- Java (Spring Boot) – Backend
- Python (Flask) – AI Service
- React – Frontend
- PostgreSQL – Database

## Architecture
Spring Boot backend communicates with a Python AI microservice via REST APIs.
